Method

Each article keeps two layers: readable page and source document.

The HTML pages are for navigation and public reading. The Markdown files are the durable source of truth, designed for version history, review, rewriting, citation, and long-term preservation.

  1. Conversation produces a theme
  2. Theme becomes Markdown source
  3. Markdown becomes article page
  4. Homepage index links the archive
  5. Git records every revision

A living map, not a finished doctrine.

The archive should evolve as new themes appear. The goal is not to freeze a theory early, but to preserve the path by which the theory becomes sharper.
